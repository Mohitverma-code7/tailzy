Build Your Own CSS Engine

A lightweight utility-first CSS engine that parses chai-* classes and applies them as inline styles. No build step. No dependencies.

  h2>
  <p class="chai-text-base chai-opacity-80">
    No build tools. Just classes.
  p>
div>

The Process

How It Works

1

Write Classes

Add chai-* utility classes directly to your HTML elements.

2

Engine Scans

ChaiTailwind scans the DOM, finds every element with a chai- prefix.

3

Styles Applied

Inline styles are computed and applied. Original classes are cleaned up automatically.
